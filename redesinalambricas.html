¿Que es una red inalambrica?

Una red inalámbrica es una red de comunicación que utiliza ondas de radio o tecnologías similares para transmitir datos entre dispositivos sin necesidad de cables. En lugar de conexiones físicas, los dispositivos se comunican a través de señales de radiofrecuencia, infrarrojas o microondas.

¿Como funciona una red inalambrica?

La seguridad de una red funciona implementando medidas para proteger la información y los dispositivos conectados. Estas medidas incluyen autenticación de usuarios, cifrado de datos, uso de firewalls para filtrar tráfico, detección de intrusos, protección contra malware y control de accesos. Además, se aplican protocolos de seguridad como SSL/TLS e IPSec para garantizar conexiones seguras y evitar ataques cibernéticos.

Tipos de redes inalambricas

  • WI-FI: Para conexiones a redes locales de alta velocidad en hogares y oficinas.
  • Bluethoot: Para comunicación de corto alcance entre dispositivos como teléfonos y auriculares.
  • Zigbee: Usado en automatización del hogar y dispositivos IoT con bajo consumo energético.
  • LTE4G/5G: Redes móviles para acceso a Internet de alta velocidad en áreas amplias.
  • Infrared: Usado en dispositivos como controles remotos con un alcance muy limitado.
  • wiMax: Tecnología de banda ancha para áreas rurales y redes de área amplia.
  • NFC: Comunicación de muy corto alcance para pagos y transferencia rápida de datos.

    • ¿Qué funcion tiene una red inalambrica?

    Funciones de una red inalambrica:

  • Conexion a internet:Facilita el acceso a Internet sin la necesidad de cables, como en redes Wi-Fi.
  • Conexion entre dispositivos: Permite la comunicación entre dispositivos cercanos, como en el caso de Bluetooth o Zigbee.
  • Mantener la disponibilidad: Garantiza que los servicios de red estén disponibles para los usuarios autorizados sin interrupciones.
  • Movilidad Los dispositivos pueden moverse libremente dentro de un área de cobertura sin perder la conexión a la red.
  • Seguridad y cifrado: Protege la información mediante firewalls, cifrado y autenticación de usuarios.
  • Facilidad de instalacion: Al no requerir cables, la instalación de una red inalámbrica es más rápida y flexible.
  • Accesibilidad:Ofrece acceso a redes locales o Internet desde cualquier lugar dentro del área de cobertura.
